📊 Data Jobs Dashboard 2.0 (Power BI)

An enhanced and redesigned Power BI dashboard analyzing 479K+ global data job postings, with improved UI, dynamic measures, and advanced filtering capabilities.

This version focuses on skills analysis, salary benchmarking, and interactive user-driven insights.

🚀 Project Overview

Data Jobs Dashboard 2.0 introduces a modern dark-themed interface and enhanced analytics for deeper job market exploration.

📌 Key Metrics

📄 Total Job Count: 479K

🛠 Average Skills per Job: 4.8

💰 Median Yearly Salary: $113K

⏱ Median Hourly Salary: $48

📊 Key Insights

Python and SQL dominate job skill requirements

Senior Data Scientist & ML Engineer roles offer the highest salaries

Skills-based analysis provides clearer market positioning

Dynamic salary toggle between yearly and hourly views

Country and job-title level filtering

🔎 Features

🎯 Job Title & Country slicers

🔄 Clear slicer functionality

📊 Skill popularity analysis

💰 Salary comparison by role

📈 Toggle between job count and job percent

📊 Median salary toggle (Yearly / Hourly)

Clean dark UI for professional presentation

🛠 Tech Stack

Power BI Desktop

Star Schema Data Modeling

DAX (Dynamic Measures & Toggles)

Power Query (Data Transformation)

🎯 Objective

To build an advanced, interactive job market intelligence dashboard that enables users to:

Analyze in-demand skills

Compare salary benchmarks

Filter by country and job role

Understand hiring trends through dynamic KPIs
